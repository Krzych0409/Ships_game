1. The game was created with player vs bot in mind, but you can easily make the algorithms duel.
All you need to do is change the third argument "kind_of_shot" when creating an object of class "Player".
2. The player can position the ships himself (Y) or use the automatic deployment option (N).
3. Both field coordinates are given in one answer. The rows are given A-J, and the columns are given 1-10. The order does not matter.


Gra została stworzona z myślą player vs bot, ale można w łatwy sposób zrobić pojedynek algorytmów.
Wystarczy tylko zmienić trzeci argument "kind_of_shot" przy tworzeniu obiektu klasy "Player".

Gracz może sam ustawiać statki (Y) lub skorzystać z opcji automatycznego rozmieszczania (N).
Obie współrzędne pola podajemy w jednej odpowiedzi. Wiersze podajemy A-J, a kolumny 1-10. Kolejność nie ma znaczenia.
